FlickPick API Documentation

This project is about building a server-side component of a “movies” web application which will provide users with access to information about different concert movies, directors and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoint Description Method URL Parameters Request Body Response Data
/users Allow new users to register POST None {"name": "string"} Created user object
/users/:id Allow users to update their user info (username) PUT id: User ID {"name": "string"} Updated user object
/users/:id/:movieTitle Allow users to add a movie to their list of favorites POST id: User ID, movieTitle: Movie Title None Success message
/users/:id/:movieTitle Allows users to delete a movie from their favorites DELETE id: User ID, movieTitle: Movie Title None Success message
/users/:id Allow existing users to deregister DELETE id: User ID None Success message
/movies Return a list of all movies GET None None List of all movies
/movies/:title Return data about a single movie by title GET title: Movie Title None Movie object
/movies/genre/:genreName Return data about a genre by name/title GET genreName: Genre Name None Genre object
/movies/directors/:directorName Return data about a director by name GET directorName: Director Name None Director object
